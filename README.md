# Brandon Chen - GitHub Pages

An English-only, responsive personal research website using the IPNL source design system: Fraunces, Mulish, IBM Plex Mono, warm ivory, ultramarine, vermilion, and dark indigo.

Main pages: Home, About, Research, Projects, Portfolio, CV, and Contact.

Individual research and project pages are driven by `assets/js/detail-data.js`. Shared identity and colours are configured in `assets/js/site-config.js`. Start with `EDIT-HERE.md`.

## Publish

Upload the contents of this folder to the root of `blxchen/blxchen.github.io`. GitHub Pages will serve `index.html` as the homepage.
